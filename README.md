## Run Locally   

Clone the project

``` console
git clone git@github.com:Harrytimbog/cronies-api.git
```

Go to the project directory

``` console
cd cronies-api
```

Install dependencies

``` console
npm install
```

Install gems

``` console
bundle install
```

Create, Migrate and Seed Database

``` console
rails db:create db:migrate db:seed
```


Start Server

``` console
rails s
```
